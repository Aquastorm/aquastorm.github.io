Project Description

{{ page.projectName }} is a 2D real-time multiplayer shooter game. Players can fight PvP battles alone or with friends in one of the three offered game modes: death match, capture the flag or battle royale.

Main Tasks

  • Developed most of the menu UI and developed a few components and screens of the in-game one.
  • Created multiple reusable and configurable controls and widgets (UI components).
  • Built many core functionalities, e.g., code used by multiple systems to locally store/read game data to/from the device's local storage, network code to access web services.
  • Developed the following systems:
    • Localization
    • Screens and popups management, navigation & animation
    • Tabs management & associated animations
    • The client-side code of a remote configuration system, to remotely drive many aspects of the game (e.g., duration and amount of player for every game mode, weapons damage and range, players walking and running speed).
  • Game profiling and use of various optimizations techniques to improve game performance.
  • Integrated the following SDKs:
    • Google Firebase SDKs: Firebase Cloud Messaging (FCM) and Firebase Dynamic Links.
    • GameAnalytics SDK to collect core and customized metrics.
  • Implemented the functionality to generate and scan QR codes to add other players to the actual list of friends.
